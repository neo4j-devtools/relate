# Usage

<!-- usage -->

```sh-session
$ npm install -g @relate/cli
$ relate COMMAND
running command...
$ relate (-v|--version|version)
@relate/cli/0.0.0 linux-x64 node-v10.18.0
$ relate --help [COMMAND]
USAGE
  $ relate COMMAND
...
```
