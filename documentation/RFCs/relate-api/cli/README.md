# Usage

<!-- usage -->
```sh-session
$ npm install -g @daedalus/cli
$ daedalus COMMAND
running command...
$ daedalus (-v|--version|version)
@daedalus/cli/0.0.0 linux-x64 node-v10.18.0
$ daedalus --help [COMMAND]
USAGE
  $ daedalus COMMAND
...
```
