# Usage

<!-- usage -->
```sh-session
$ npm install -g @relate/cli
$ relate COMMAND
running command...
$ relate (-v|--version|version)
@relate/cli/1.0.0 linux-x64 node-v12.16.0
$ relate --help [COMMAND]
USAGE
  $ relate COMMAND
...
```
